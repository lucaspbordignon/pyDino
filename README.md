# pyDino
A simple game.

## Usage
```
$ pip install -r requirements.txt
$ ./init
```
## How to play
In the choose character menu, you have to click in the number and the dino. <br>
-Up key to jump <br>
-Down key to go down <br>
-H to shoot(cost 5 coins)

## Coins
-Ether = 1 coin <br>
-Bitcoin = 2 coins <br>
-Dogecoin = 4 coins(such money)
